Q11 Write a function that accepts parameter n and returns factorial of n

Ans

                unction factorial(n) //take argument as parameter
        {
            let fact = 1;
            while (n!=0) 
                {
                    for (let i=1 ; i<=n; i++)
                    {
                        fact = fact *i; //from i=1 to n loop iterate and everytime fact update and when for loop condition false control come out of for loop body
      
                    }
                    n=0;    //for stop while loop after completing factorial of number n
                }
            return fact;
        }
    const num = factorial(4); //passing 4 as argument 
    console.log(`factorial of 4=`,num)

Reason : n=0; for stop while loop after completing factorial of number n inside for loop
fact = fact *i; from i=1 to n loop iterate and everytime fact update and when for loop condition false control come out of for loop body
